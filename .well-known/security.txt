# Coordinated Vulnerability Disclosure (CVD) Process
#
# This file follows RFC 9116: https://www.rfc-editor.org/rfc/rfc9116.html
#
# Please report security vulnerabilities through:
# 1. Email: security@eclexia.dev
# 2. GitHub Security Advisories (preferred for tracking)
# 3. Encrypted: PGP key available on request
#
# Response Timeline:
# - Acknowledgment: Within 48 hours
# - Initial Assessment: Within 7 days
# - Status Updates: Every 14 days
# - Coordinated Disclosure: 90 days after fix
#
# What to include in your report:
# - Description of the vulnerability
# - Steps to reproduce
# - Affected versions
# - Potential impact
# - Suggested remediation (if known)
#
# We follow responsible disclosure practices and work with
# security researchers to address issues before public disclosure.
#
# Security researchers will be credited in our Security Hall of Fame
# unless anonymity is requested.
#
# Thank you for helping keep Eclexia and its users safe!
